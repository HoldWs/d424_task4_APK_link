This page is being used to host the APK for Cole McDougal's Software Engineering Capstone project.

The developed app for this assignment allows users to schedule vacations and excursions through a mobile interface. Users can customize their trips with a variety of details and share their trips with friends. Using the Room framework allows for adding, updating and deleting from a SQLite database.

The user is also able to search for specific vacations with a search tool and generate a report of scheduling activity within the app.
